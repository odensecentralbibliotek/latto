Mixins available

Latto basetheme use various mixins for making life easy when writing CSS. find them all in the file sass/_latto.mixins.scss

rem()

Mixin that converts pixel values to rem values for whatever property is passed to it. Returns two lines of code - one of the regular pixel values (for older browsers) and another with the converted rem values (for everyone else).

Basic usage:
@include rem('padding',10px 0 2px 5px);
@include rem('font-size',10px);



input-icons-inside()

Candidate for a name change.

This mixins positions a button and an icons inside an input field, calculating the position of the icon and button from the $height parameter.

All the elements must be placed within a wrapper element, which includes this mixin.

Icons must be of in I element and button must have 'btn' class.

NOTE: If the an icon is included, the input field must have 'indent' in class name.

Basic usage:
@include input-icons-inside();
@include input-icons-inside(50px);
