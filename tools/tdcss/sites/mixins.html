Mixins available

Latto basetheme use various mixins for making life easy when writing CSS. find them all in the file sass/_latto.mixins.scss

rem()

Mixin that converts pixel values to rem values for whatever property is passed to it. Returns two lines of code - one of the regular pixel values (for older browsers) and another with the converted rem values (for everyone else).

Basic usage:
@include rem('padding',10px 0 2px 5px);
@include rem('font-size',10px);
